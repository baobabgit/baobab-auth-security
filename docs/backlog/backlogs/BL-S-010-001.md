# BL-S-010-001 — Initialiser le package

:version_target: v0.1.0
:priority: P0
:depends_on: []

## Contexte

Le dépôt part d'un template (`example_package`). Cette tâche transforme le
squelette en package `baobab_auth_security` (layout `src/`), câble la dépendance
runtime `baobab-auth-core`, et pose le socle technique commun : `version.py`,
`py.typed`, hiérarchie d'exceptions, horloge injectable.
