# BL-S-010-002 — Implementer le hash de mot de passe Argon2id

:version_target: v0.1.0
:priority: P0
:depends_on: [BL-S-010-001]

## Contexte

Fournir le hachage et la vérification de mot de passe en Argon2id, base de
l'inscription et du login côté API consommatrice.

## FEAT liées

- FEAT-010.1 — Hachage de mot de passe Argon2id (US-010)

## Critères d'acceptation

- [ ] `PasswordHashPolicy` (dataclass figée) encapsule les paramètres Argon2.
- [ ] `Argon2PasswordHasher.hash` retourne un `PasswordHashResult` non vide,
      non réversible, préfixé `$argon2id$`.
- [ ] `verify` retourne `PasswordVerificationResult` (booléen) sans lever sur
      mauvais mot de passe.
- [ ] `needs_rehash` détecte un changement de paramètres.
- [ ] Aucun secret exposé dans `repr`/`str`.
